Creating your own

Creating your own is quite simple; only a bit of JavaScript (or, as some people know it, EMCAScript) is needed.

WARNING: Your documents WILL NOT be valid HTML!

Importing BetterElement

Importing BetterElement is quite easy. The following code will make it work:

function loadMyPlugin(){
var script = document.createElement("script");
script.src = "https://raw.githubusercontent.com/wapidstyle/wapi/master/js/betterelement.js";
script.type = "text/javascript";
document.getElementsByTagName("head")[0].appendChild(script);
}

Creating an Element

We have BetterElement... for what? We should actually add an element. Here's the code to make a <hello> tag:

function doHello(){
var hellotag = createTag("hello");
}

This will return an array of all the <hello> tags on the page. Here's how to use it:

for(var count = 0; hellotag[count] != null; count = count + 1){
hellotag[count].innerHTML = "Hello World!"
}

This makes every <hello> tag on the page say "Hello World!" inside the <hello> tags.

Now, save that. It does not matter what you name, however it must be saved as JavaScript (.js).

Of course, you can rename loadMyPlugin() the name of your plugin. For example, loadEveri() would work for one named Everi. You also, if you wish, don"t need to include the load in the declaration. You can use something else, or simply leave it out. It won"t make a change.

Adding Attributes

Adding attributes is quite easy. You can quite easily make a <text> tag with a text attribute. Using <text text="Hi Everyone!">, with this setup, would create:

Hi Everyone!

Here's how to do it:

  1. Make a tag. Scroll up until you find "Creating an Element" if you are unsure how to do this.
  2. Ensure the text attribute is not empty or simply not there. You can do this through the following if gate:
    if(mytag[count].getAttribute("text") != undefined && mytag[count].getAttribute("text") != ""){}
  3. Get the text attribute and set the inner HTML of the element to the text value. Put this inside step 2's if gate:
    mytag[count].innerHTML = mytag[count].getAttribute("text");
    This will make any <text> elements display their text attribute.
  4. Report an error if the element is missing the text element. You can do this by adding, next to to the last } on step 2's if gate:
    else{console.error("Text element #" + count + " missing text attribute!")}
    This will cause any text attributes missing from text elements to be reported to the console. Stop those mistakes! :P

And now you're done!

Importing your script

Now, we need to make the script load. If we simply leave the code there, it"s a waste of memory. All you need to do is import your script. This can be accompished by putting the following in the HTML page:

<script>
loadMyPlugin();
</script>

That's it to making your own elements!
