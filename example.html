BetterElement

BetterElement lets you create custom tags in JavaScript. You can then use these in your HTML with attributes.

Creating your own

You can create your own by importing BetterElement and utilising the Element class to create and read elements. To learn how, visit https://wapidstyle.github.io/wapi/subprojects/betterelement.html.
